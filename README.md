## Installation

django-secure-passwords is currently available only on Blueshoe's Python Package Index.
```bash
pip3 install django-secure-passwords
```

Add *"django-secure-passwords"* to your *INSTALLED_APPS*: 
```python
INSTALLED_APPS = [
    "...",
    "securepasswords",
]
```
